# Manhattan
Manhattan - адаптивный кроссбраузерный макет сайта
## Стек технологий

+ HTML5
+ SASS
+ GULP
+ VUE
+ JAVASCRIPT
+ SVG

## Установка

+ Клонировать проект

```
git clone https://github.com/GaynetdinovR/Manhattan.git
```

+ Установить необходимые пакеты

```
npm install
```

+ Запустить таск-менеджер

```
npm run gulp
```

+ Запустить сервер

```
npm run serve
```
